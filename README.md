#  Password Strength Checker

A Python-based cybersecurity tool that analyzes password strength, checks for common weaknesses, estimates password complexity, and suggests stronger passwords.

## Features

- Checks password length
- Detects uppercase, lowercase, numbers, and special characters
- Identifies common weak patterns
- Gives a strength rating: Very Weak, Weak, Moderate, Strong, Very Strong
- Calculates a simple entropy estimate
- Suggests how to improve the password

## Tech Stack

- Python 3

## How to run

```bash
python password_checker.py
